== Target

[%autowidth,cols="1,1",options="header"]
|===
|Item |Setting
|Style |Modern metalcore/deathcore bass that supports dry high-gain guitars
|Core formula |Clean compressed low end plus distorted mid/high grind
|Low path role |Stable sub and punch with no distortion
|Dirty path role |Growl, pick/finger definition, controlled clank
|Main IR recommendation |York Audio AMPG 410 HS, 48k Mix file
|Output |Mono, centered
|===

The bass should supply the true weight below the guitars and add grind around 700 Hz to 2.5 kHz so it remains audible through the band.

== Frequency priority with the guitars

[%autowidth,cols="1,1",options="header"]
|===
|Frequency area |Instrument priority
|35 to 60 Hz |Bass low foundation
|70 to 100 Hz |Bass punch and kick relationship
|120 to 250 Hz |Controlled, not excessive
|700 Hz to 1.2 kHz |Bass growl
|1.4 to 2.2 kHz |Bass note definition
|2.5 to 4 kHz |Pick/clank, carefully controlled
|Above 6.5 kHz |Usually unnecessary for bass
|===

If the guitars sound smaller when the bass enters, the bass likely has too much 150 to 300 Hz or too much distorted low end below 180 Hz.
