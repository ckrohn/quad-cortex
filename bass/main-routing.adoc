== Main split chain

Use a parallel split. Path A is clean low end. Path B is dirty mid/high grind.

[source,text]
----
In 1
-> VCA Comp
-> Splitter: Crossover

Path A, clean low path:
Low-High Cut -> VCA Comp -> Parametric-8 -> Gain -> Mixer

Path B, dirty mid/high path:
Low-High Cut -> Microtubes B3K -> Amped Super Valve -> IR Loader: Single (M)
-> Adaptive Gate (S/C) -> Low-High Cut -> Parametric-8 -> Gain -> Mixer

Mixer
-> Parametric-8 -> Gain -> Out 1 / Multi Out
----

== In 1

[%autowidth,cols="1,1,1",options="header"]
|===
|Parameter |Passive bass |Active bass
|Input Gain |0.0 dB |-6.0 dB
|Input Gate |Off |Off
|===

Use the active-bass setting if the player has a hot preamp, active EMGs, Fishman bass pickups, or anything that clips the input on hard picked notes.

== Input adjustment rules

[%autowidth,cols="1,1",options="header"]
|===
|Behavior |Change
|Input flashes red on hard notes |Lower Input Gain by 3 dB
|Bass feels weak and never gets near yellow |Raise Input Gain by 3 dB
|Low notes distort before the drive path |Lower Input Gain, not the amp gain
|===

== Pre-split VCA Comp

[%autowidth,cols="1,1",options="header"]
|===
|Parameter |Setting
|Threshold |-20.0 dB
|Ratio |4.0:1
|Attack |18 ms
|Release |90 ms
|Makeup / Output |+2.0 dB
|Mix |75%
|===

Aim for roughly 3 to 6 dB of gain reduction on hard notes.

== Pre-split compressor fine-tuning

[%autowidth,cols="1,1",options="header"]
|===
|Problem |Change
|Pick attack disappears |Raise Attack to 25 ms
|Low notes jump out too much |Lower Threshold to -24 dB
|Bass feels over-compressed |Raise Threshold to -16 dB, Mix 65%
|Bass is still uneven |Ratio 5.0:1, Release 110 ms
|===

== Splitter: Crossover

[%autowidth,cols="1,1",options="header"]
|===
|Parameter |Setting
|Type |Crossover
|Frequency |180 Hz
|Mode |Normal / not swapped
|Stereo |Off
|Mute |Off
|===

== Crossover frequency guide

[%autowidth,cols="1,1",options="header"]
|===
|Tuning / role |Frequency
|B standard / Drop A |180 Hz
|Very low tuning, Drop G or lower |150 to 165 Hz
|Bass is too clean and separated |200 Hz
|Bass distortion muddies the guitars |190 to 220 Hz
|===
