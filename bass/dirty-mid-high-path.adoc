== Purpose

This path provides bass aggression. It should sound ugly and somewhat thin by itself. In the full preset, it gives note definition through the two guitars.

[source,text]
----
Path B:
Low-High Cut -> Microtubes B3K -> Amped Super Valve -> IR Loader: Single (M)
-> Adaptive Gate (S/C) -> Low-High Cut -> Parametric-8 -> Gain -> Mixer
----

== Pre-drive Low-High Cut

[%autowidth,cols="1,1",options="header"]
|===
|Parameter |Setting
|Low Cut |180 Hz
|High Cut |7.5 kHz
|===

== Pre-drive cut adjustments

[%autowidth,cols="1,1",options="header"]
|===
|Problem |Change
|Distortion makes the low end fuzzy |Raise Low Cut to 200 Hz
|Distorted bass feels disconnected from lows |Lower Low Cut to 160 Hz
|Too much pick scratch |Lower High Cut to 6.2 kHz
|Not enough clank |Raise High Cut to 8.5 kHz
|===

== Microtubes B3K fine-tuning

[%autowidth,cols="1,1",options="header"]
|===
|Problem |Change
|Too much fizz |Tone 4.0, Drive 5.0
|Not enough aggression |Drive 6.5, Tone 5.3
|Bass masks the guitars |Lower Blend to 5.5
|Bass disappears |Keep Drive the same and raise dirty path level
|Palm-muted guitar riffs lose tightness |Raise Path B Low Cut to 200 Hz
|Bass sounds like fuzz instead of grind |Drive 4.8, Blend 8.0
|===

== Adaptive Gate (S/C)

[%autowidth,cols="1,1",options="header"]
|===
|Parameter |Setting
|Noise Reduction |42.0%
|Source / Trigger |In 1
|===

Use this only to clean up hiss from the distorted path. Do not clamp the bass like a rhythm guitar.

== Gate fine-tuning

[%autowidth,cols="1,1",options="header"]
|===
|Problem |Change
|Hiss between stops |Noise Reduction 50%
|Sustained bass notes die |Noise Reduction 30% to 35%
|Bass feels disconnected |Turn the gate off
|Dead stops are too loose |Check muting technique first, then try 55%
|===
