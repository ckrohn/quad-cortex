== Phase check

Play hard low-string palm mutes with the full band at rehearsal volume.

[%autowidth,cols="1,1",options="header"]
|===
|Result |Setting
|Low end gets bigger and tighter with Phase Off |Keep Phase Off
|Low end gets smaller when paths are blended |Turn Phase On
|Difference is minor |Use the setting with more punch around 70 to 120 Hz
|===

== What the post-mix EQ does

[%autowidth,cols="1,1",options="header"]
|===
|Area |Function
|55 to 85 Hz |Main low-end mass
|130 Hz |Prevents low-end bloom
|240 Hz |Main mud control
|500 Hz |Removes boxiness
|900 Hz |Bass growl
|1.60 kHz |Note definition through guitars
|3.80 kHz |Controls scratch
|===

== Final Gain

[%autowidth,cols="1,1",options="header"]
|===
|Parameter |Setting
|Gain |Preset volume trim, usually -4.0 dB to 0.0 dB
|===

Use this final Gain block for preset volume matching. Do not use compressor output, amp output, or dirty path level as the master volume.
