= Quad Cortex Bass Preset - Split Modern Metal Bass
:toc: macro
:toclevels: 4
:sectanchors:
:icons: font

Split bass preset designed to complement the 6505 and EV101 guitar tones.

This folder now uses a single-file layout. All bass routing, path settings, scenes, variants, quick-adjustment tables, and final summary are contained in this `README.adoc`.

toc::[]

[[target]]
== Target

[%autowidth,cols="1,1",options="header"]
|===
|Item |Setting
|Style |Modern metalcore/deathcore bass that supports dry high-gain guitars
|Core formula |Clean compressed low end plus distorted mid/high grind
|Low path role |Stable sub and punch with no distortion
|Dirty path role |Growl, pick/finger definition, controlled clank
|Main IR recommendation |York Audio AMPG 410 HS, 48k Mix file
|Output |Mono, centered
|===

The bass should supply the true weight below the guitars and add grind around 700 Hz to 2.5 kHz so it remains audible through the band.

[[frequency-priority]]
== Frequency priority with the guitars

[%autowidth,cols="1,1",options="header"]
|===
|Frequency area |Instrument priority
|35 to 60 Hz |Bass low foundation
|70 to 100 Hz |Bass punch and kick relationship
|120 to 250 Hz |Controlled, not excessive
|700 Hz to 1.2 kHz |Bass growl
|1.4 to 2.2 kHz |Bass note definition
|2.5 to 4 kHz |Pick/clank, carefully controlled
|Above 6.5 kHz |Usually unnecessary for bass
|===

If the guitars sound smaller when the bass enters, the bass likely has too much 150 to 300 Hz or too much distorted low end below 180 Hz.

[[main-routing]]
== Main Routing

=== Main split chain

Use a parallel split. Path A is clean low end. Path B is dirty mid/high grind.

[source,text]
----
In 1
-> VCA Comp
-> Splitter: Crossover

Path A, clean low path:
Low-High Cut -> VCA Comp -> Parametric-8 -> Gain -> Mixer

Path B, dirty mid/high path:
Low-High Cut -> Microtubes B3K -> Amped Super Valve -> IR Loader: Single (M)
-> Adaptive Gate (S/C) -> Low-High Cut -> Parametric-8 -> Gain -> Mixer

Mixer
-> Parametric-8 -> Gain -> Out 1 / Multi Out
----

=== In 1

[%autowidth,cols="1,1,1",options="header"]
|===
|Parameter |Passive bass |Active bass
|Input Gain |0.0 dB |-6.0 dB
|Input Gate |Off |Off
|===

Use the active-bass setting if the player has a hot preamp, active EMGs, Fishman bass pickups, or anything that clips the input on hard picked notes.

=== Input adjustment rules

[%autowidth,cols="1,1",options="header"]
|===
|Behavior |Change
|Input flashes red on hard notes |Lower Input Gain by 3 dB
|Bass feels weak and never gets near yellow |Raise Input Gain by 3 dB
|Low notes distort before the drive path |Lower Input Gain, not the amp gain
|===

=== Pre-split VCA Comp

[%autowidth,cols="1,1",options="header"]
|===
|Parameter |Setting
|Threshold |-20.0 dB
|Ratio |4.0:1
|Attack |18 ms
|Release |90 ms
|Makeup / Output |+2.0 dB
|Mix |75%
|===

Aim for roughly 3 to 6 dB of gain reduction on hard notes.

=== Pre-split compressor fine-tuning

[%autowidth,cols="1,1",options="header"]
|===
|Problem |Change
|Pick attack disappears |Raise Attack to 25 ms
|Low notes jump out too much |Lower Threshold to -24 dB
|Bass feels over-compressed |Raise Threshold to -16 dB, Mix 65%
|Bass is still uneven |Ratio 5.0:1, Release 110 ms
|===

=== Splitter: Crossover

[%autowidth,cols="1,1",options="header"]
|===
|Parameter |Setting
|Type |Crossover
|Frequency |180 Hz
|Mode |Normal / not swapped
|Stereo |Off
|Mute |Off
|===

=== Crossover frequency guide

[%autowidth,cols="1,1",options="header"]
|===
|Tuning / role |Frequency
|B standard / Drop A |180 Hz
|Very low tuning, Drop G or lower |150 to 165 Hz
|Bass is too clean and separated |200 Hz
|Bass distortion muddies the guitars |190 to 220 Hz
|===

== Path A - Clean Low Path

=== Purpose

This path is the clean foundation. It should sound stable, controlled, centered, and relatively plain by itself.

[%autowidth,cols="1,1",options="header"]

[[path-b-dirty-mid-high-path]]
== Path B - Dirty Mid/High Path

=== Purpose

This path provides bass aggression. It should sound ugly and somewhat thin by itself. In the full preset, it gives note definition through the two guitars.

[source,text]
----
Path B:
Low-High Cut -> Microtubes B3K -> Amped Super Valve -> IR Loader: Single (M)
-> Adaptive Gate (S/C) -> Low-High Cut -> Parametric-8 -> Gain -> Mixer
----

=== Pre-drive Low-High Cut

[%autowidth,cols="1,1",options="header"]
|===
|Parameter |Setting
|Low Cut |180 Hz
|High Cut |7.5 kHz
|===

=== Pre-drive cut adjustments

[%autowidth,cols="1,1",options="header"]
|===
|Problem |Change
|Distortion makes the low end fuzzy |Raise Low Cut to 200 Hz
|Distorted bass feels disconnected from lows |Lower Low Cut to 160 Hz
|Too much pick scratch |Lower High Cut to 6.2 kHz
|Not enough clank |Raise High Cut to 8.5 kHz
|===

=== Microtubes B3K

[%autowidth,cols="1,1",options="header"]
|===
|Parameter |Setting
|Drive |5.8
|Growl |On
|Midboost |On
|Tone |4.8
|Level |5.0
|Blend |7.0
|===

=== Microtubes B3K fine-tuning

[%autowidth,cols="1,1",options="header"]
|===
|Problem |Change
|Too much fizz |Tone 4.0, Drive 5.0
|Not enough aggression |Drive 6.5, Tone 5.3
|Bass masks the guitars |Lower Blend to 5.5
|Bass disappears |Keep Drive the same and raise dirty path level
|Palm-muted guitar riffs lose tightness |Raise Path B Low Cut to 200 Hz
|Bass sounds like fuzz instead of grind |Drive 4.8, Blend 8.0
|===

=== Amped Super Valve

[%autowidth,cols="1,1",options="header"]

|Midrange |6.5

|Ultra Hi |Off
|Master |4.0

=== Amp fine-tuning

[%autowidth,cols="1,1",options="header"]

|Bass grind is too woolly |Bass 1.0, Midrange 7.0

|Too much scratch |Treble 3.5, B3K Tone 4.2
|Too dark |Treble 4.8, B3K Tone 5.4
|Guitars feel smaller when bass enters |Bass 1.5, raise Path B Low Cut to 200 Hz
|===

=== IR Loader: Single (M)

[%autowidth,cols="1,1",options="header"]

[%autowidth,cols="1,1,1",options="header"]

|OwnHammer AMPG Bass |York 410 is too big or extended |Start with a 410HLF mix, not a 210-only IR

=== Mixer settings

[%autowidth,cols="1,1",options="header"]

|Pan B |C

|===

=== Phase check

Play hard low-string palm mutes with the full band at rehearsal volume.

[%autowidth,cols="1,1",options="header"]
|===
|Result |Setting
|Low end gets bigger and tighter with Phase Off |Keep Phase Off
|Low end gets smaller when paths are blended |Turn Phase On
|Difference is minor |Use the setting with more punch around 70 to 120 Hz
|===

|3 |Bell |130 Hz |-1.0 dB |1.00
|4 |Bell |240 Hz |-2.5 dB |1.10
|5 |Bell |500 Hz |-1.5 dB |1.00

=== What the post-mix EQ does

[%autowidth,cols="1,1",options="header"]
|===
|Area |Function
|55 to 85 Hz |Main low-end mass
|130 Hz |Prevents low-end bloom
|240 Hz |Main mud control
|500 Hz |Removes boxiness
|900 Hz |Bass growl
|1.60 kHz |Note definition through guitars
|3.80 kHz |Controls scratch
|===

=== Final Gain

[%autowidth,cols="1,1",options="header"]
|===
|Parameter |Setting
|Gain |Preset volume trim, usually -4.0 dB to 0.0 dB
|===

Use this final Gain block for preset volume matching. Do not use compressor output, amp output, or dirty path level as the master volume.

[[quick-adjustments]]
== Quick Adjustments

=== Main correction table

[%autowidth,cols="1,1",options="header"]
|===
|Problem |Fix
|Bass makes palm-muted guitars muddy |Raise Path B Low Cut to 200 Hz and cut final 240 Hz another -1 dB
|Bass has weight but no note definition |Raise Path B 1.15 kHz and 1.85 kHz by +1 dB
|Bass sounds like a third guitar |Lower Path B Level to -5 dB and lower B3K Blend to 5.5
|Bass disappears under guitars |Raise Path B Level to -2.5 dB and boost 900 Hz and 1.6 kHz
|Low B / A is too boomy |Lower Path A 70 Hz boost and cut 105 Hz another -1 dB
|Low end is weak |Raise Path A Level +1 dB and boost 70 to 85 Hz
|Bass is too clicky |Lower B3K Tone to 4.0 and cut 2.9 kHz
|Bass is too dark |Raise B3K Tone to 5.5 and raise Path B Level
|Stops are noisy |Raise Adaptive Gate to 50%
|Sustains get cut off |Lower Adaptive Gate to 30% or bypass it
|Bass and guitars fight in low mids |Cut final 240 Hz and Path B 220 Hz
|===
