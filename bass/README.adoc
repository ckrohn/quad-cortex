== Target

This bass preset is designed to complement the two dry modern guitar tones. The core idea is simple: keep the low end clean and compressed, then add distorted mid/high grind on a separate path.

[source,text]
----
clean compressed low end
+ distorted mid/high grind
+ controlled upper clank
+ no distorted sub-low
----

The guitars are cut and tightened around the low end. The bass should supply the actual weight while still being audible through dense guitar rhythm parts.

== Full routing

[source,text]
----
In 1
-> VCA Comp
-> Splitter: Crossover

Path A, clean low path:
Low-High Cut
-> VCA Comp
-> Parametric-8
-> Gain
-> Mixer

Path B, dirty mid/high path:
Low-High Cut
-> Microtubes B3K
-> Utility Gate (optional, default bypassed)
-> Amped Super Valve
-> IR Loader: Single (M)
-> Low-High Cut
-> Parametric-8
-> Gain
-> Mixer

Mixer
-> Parametric-8
-> Gain
-> Output
----

== Input settings

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|Input Gain, passive bass |0.0 dB |A passive bass usually needs more input level than an active bass. Use the meter as the final judge.
|Input Gain, active bass |-6.0 dB |Active bass preamps can clip the input on hard picked notes. Start lower to keep the low end clean.
|Input Gate |Off |The clean low path should not be gated. Muting should come from the player and from the dirty path gate only if needed.
|===


If the input flashes red on hard notes, lower Input Gain by `3 dB`. If the bass never reaches a healthy level, raise it by `3 dB`.

== Pre-split VCA Comp

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|Threshold |-20.0 dB |Catches loud peaks before the signal is split. Adjust so hard notes produce roughly 3 to 6 dB of gain reduction.
|Ratio |4.0:1 |Strong enough for metal bass consistency, not so strong that every note becomes flat.
|Attack |18 ms |Lets some pick or finger transient through before compression clamps down.
|Release |90 ms |Fast enough to recover between notes, slow enough to avoid obvious pumping.
|Makeup / Output |+2.0 dB |Restores some level after compression.
|Mix |75% |Parallel compression feel: mostly controlled, still somewhat natural.
|===


This compressor evens out the whole instrument before the split. The low path gets additional compression later, but this first block stabilizes performance dynamics.

== Splitter: Crossover

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|Type |Crossover |Splits lows and mids/highs so only the upper range is distorted.
|Frequency |180 Hz |Keeps the true lows clean and routes grind-relevant mids to the dirty path.
|Mode |Normal, not swapped |Path A receives lows and Path B receives highs.
|Stereo |Off |Mono bass is more stable live and in the mix.
|Mute |Off |Both paths feed the Mixer. Balance is handled with path Gain and Mixer levels.
|===


Use `150` to `165 Hz` for very low tunings such as Drop G if the dirty path feels disconnected. Use `190` to `220 Hz` if the distortion is making palm-muted guitars muddy.

== Path A - clean low path

Path A should sound clean, compressed, centered, and somewhat boring by itself. That is correct. It carries the low foundation while Path B supplies audible character.

=== Path A Low-High Cut

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|Low Cut |35 Hz |Removes sub-rumble below the useful bass range while preserving low B and low A weight.
|High Cut |180 Hz |Keeps this path strictly low so midrange grind comes only from Path B.
|===


Use `40` to `45 Hz` in small rooms or on weak PAs. Use `30` to `32 Hz` only if the tuning and PA can support it cleanly.

=== Path A VCA Comp

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|Threshold |-26.0 dB |More compression than the pre-split comp because low frequencies need stricter control.
|Ratio |5.0:1 |Keeps low notes consistent without hard limiting.
|Attack |30 ms |Lets the front of the note breathe so the bass does not feel slow.
|Release |130 ms |Holds the low end stable between notes.
|Makeup / Output |+3.0 dB |Restores level after heavier low-band compression.
|Mix |85% |Mostly compressed low path for consistency.
|===


If the low end pumps, raise Release to `170 ms`. If the low end feels slow, lower Release to `100 ms`.

=== Path A Parametric-8

[%autowidth,cols="1,1,1,1,1,3",options="header"]
|===
|Band |Type |Frequency |Gain |Q |Purpose
|1 |Bell |45 Hz |+1.0 dB |0.80 |Adds sub-body without overemphasizing unusable rumble.
|2 |Bell |70 Hz |+1.5 dB |0.90 |Main low-end punch area for the bass.
|3 |Bell |105 Hz |-1.0 dB |1.00 |Controls bloom that can make low notes too wide.
|4 |Bell |145 Hz |-1.8 dB |1.10 |Reduces overlap with guitar low cuts and bass cabinet warmth.
|5 |Bell |180 Hz |-2.0 dB |1.20 |Keeps crossover-area mud under control.
|6 |Bell |250 Hz |-3.0 dB |1.20 |Prevents the clean low path from clouding palm-muted guitars.
|7 |Bell |500 Hz |-6.0 dB |0.70 |Strong cut because this path is not responsible for mids.
|8 |Bell |1.00 kHz |-8.0 dB |0.70 |Removes any upper leakage from the clean-low path.
|===


=== Path A Gain

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|Gain |0.0 dB |Start neutral. Raise only if the bass lacks foundation after the dirty path is balanced.
|===


== Path B - dirty mid/high path

Path B supplies growl, pick definition, and clank. It should sound too thin by itself. If it sounds like a full bass tone alone, it likely contains too much low end.

=== Path B pre-drive Low-High Cut

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|Low Cut |180 Hz |Prevents the drive from distorting sub-low frequencies. This is the main reason the bass stays tight.
|High Cut |7.5 kHz |Removes unnecessary top before distortion while leaving enough clank.
|===


Raise Low Cut to `200 Hz` if the dirty path makes palm-muted guitars muddy. Lower to `160 Hz` if the dirty path feels disconnected from the clean lows.

=== Microtubes B3K

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|Drive |5.8 |Main grind amount. High enough for metal bass, not maxed into undefined fuzz.
|Growl |On |Adds low-mid aggression and density to the driven path.
|Midboost |On |Keeps the bass audible through two high-gain guitars.
|Tone |4.8 |Slightly dark to control active string clank and fizz.
|Level |5.0 |Neutral operating level into the amp block.
|Blend |7.0 |High blend works here because the clean low path already supplies clean fundamentals.
|===


Do not max Drive. More drive often makes bass less audible because the note center collapses into fizz.

=== Amped Super Valve

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|Gain |4.2 |Adds amp-like density after the B3K without turning the dirty path into mush.
|Bass |2.0 |Low because this path should not own the lows.
|Midrange |6.5 |Main growl control for audible bass in the guitar wall.
|Frequency |3.0 |Centers the mid focus in a useful grind range.
|Treble |4.2 |Adds presence but avoids excessive scratch.
|Ultra Lo |Off |Too much low boost on the dirty path will fight the clean-low path and guitars.
|Ultra Hi |Off |Avoids exaggerated top-end hiss and string noise.
|Master |4.0 |Moderate body and feel.
|Output / Level |Unity, usually -3.0 dB to 0.0 dB |Use for block gain staging only.
|===


The amp block makes the pedal distortion feel more like a miked bass rig. It is not intended to add large amounts of low end.

=== IR Loader: Single (M)

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|IR |York Audio AMPG 410 HS, *Mix 05* |Use the SM7 + M88 mix as the default dirty-path bass cabinet voice. It gives punch, growl, and controlled top end without adding tweeter scratch.
|Folder / format |MPT / 48k |Use the 48 kHz minimum-phase file for predictable alignment and easy loading in the Quad Cortex IR Loader.
|Mics in the mix |Vintage Shure SM7 + Beyerdynamic M88 TG |The SM7 gives smooth driven mids and restrained top end. The M88 adds punch, focus, and bass-cab authority.
|Level |0.0 dB |Neutral IR level.
|Pan |C |Mono bass keeps low end stable.
|Bypass |Off |The dirty path needs a cab/IR voice so it does not sound like raw DI distortion.
|===


Use the IR only on the dirty path. The clean low path stays direct so the low end remains extended and stable.

==== Exact York Audio AMPG 410 HS mix choice

Use this file as the default starting point:

[source,text]
----
York Audio AMPG 410 HS
Folder: MPT / 48k
Mix: Mix 05
Mics: SM7 + M88
File: the WAV file containing `Mix 05` in the MPT / 48k folder
----

Choose *Mix 05* because this preset already creates the low foundation with the clean DI-style low path. The IR should therefore make the distorted path sound like a real miked bass cabinet without adding excessive sub-low weight. The SM7 side keeps the B3K and amp drive from becoming scratchy. The M88 side adds the focused punch that makes picked or aggressively played notes stay attached to the clean low end.

Do not start with a port/tweeter-heavy mix for this preset. The dirty path already has enough clank from the B3K, the amp, and the post-drive EQ. A brighter mix can work for isolated bass, but in this band context it is more likely to fight cymbals, guitar pick attack, and vocal intelligibility.

Useful alternatives:

[%autowidth,cols="1,3,3",options="header"]
|===
|Mix |Use when |Expected change
|Mix 05, SM7 + M88 |Default |Best balance of punch, grind, and controlled top for the split modern metal preset.
|Mix 11, 57 + 160 |The bass still lacks upper-mid cut |More bite and string definition. Use carefully because it can make the dirty path feel more guitar-like.
|Mix 01, M88 + F47 |The dirty path feels too narrow or too thin |Bigger and smoother. May need a slightly higher Path B Low Cut if it adds too much low-mid body.
|Mix 08, 906 + 313 |The bassist wants more clank and edge |Brighter and more aggressive. Lower B3K Tone or the Path B High Cut if it gets scratchy.
|===

=== Optional Path B Utility Gate

Default recommendation: do *not* use Adaptive Gate (S/C) on this bass preset. Leave the clean low path ungated, and only gate the dirty mid/high path if the B3K and amp chain create enough hiss to be a problem in stops.

Place the gate here:

[source,text]
----
Microtubes B3K
-> Utility Gate
-> Amped Super Valve
----

This position is deliberate. The B3K creates a stronger and more consistent signal for the gate to read, and the Utility Gate then prevents B3K hiss and string handling noise from being pushed into the amp model. Placing the gate before the B3K usually leaves more pedal hiss behind. Placing a hard gate after the amp or IR can make the dirty bass attack and harmonic layer feel disconnected from the clean low path.

==== Default state

[%autowidth,cols="1,1,3",options="header"]
|===
|Scene |Utility Gate state |Reason
|Main rhythm |Bypassed by default |Bass sustain and note length matter more than absolute silence. The clean low path remains natural.
|Tight staccato / breakdown |On |Use only when dead stops need extra cleanup.
|Bigger chorus / open parts |Bypassed |Open bass notes should sustain naturally.
|===

==== Utility Gate settings for tight staccato / breakdown

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|Threshold |-57.0 dB |High enough to clean up dirty-path hiss and scrape noise, but not as aggressive as a guitar rhythm gate.
|Attack |1 ms |Keeps the start of the bass note immediate. Slower attack can feel like a fade-in.
|Hold |25 ms |Prevents chatter between fast picked notes.
|Release |75 ms |Fast enough for tight stops, slow enough to avoid obvious chopping.
|Range |-45.0 dB |Reduces the dirty path substantially without making the bass vanish. The clean low path is still present.
|===

==== Gentler Utility Gate settings if the dirty path is noisy in main rhythm

Use this only if the main rhythm scene has audible hiss between phrases:

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|Threshold |-62.0 dB |Lower threshold so normal sustain, harmonics, and lighter notes remain intact.
|Attack |1 ms |Keeps attack immediate.
|Hold |35 ms |Stabilizes the gate on sustained notes.
|Release |110 ms |More natural decay than the tight-staccato setting.
|Range |-35.0 dB |A partial reduction rather than a hard mute. This is usually enough for bass dirty-path cleanup.
|===

If harmonics or note definition disappear, bypass this gate before changing EQ. If the bass player has good muting and the dirty path is not noisy, the best setting is simply bypassed.

[%autowidth,cols="1,1,3",options="header"]
|===

|Level B |-3.5 dB |Dirty path supports definition without overpowering the low path.
|Pan B |C |Mono grind stays attached to the bass low end.

=== Phase check

Play hard low-string notes with both paths on. If the low end becomes smaller when both paths are blended, flip Mixer Phase and keep the setting that produces more punch around `70` to `120 Hz`. Do this at rehearsal volume.

== Optional stock-only simplified version

Use this only if the bassist does not want parallel routing. It is easier to build but less mix-ready than the split version because the same drive path affects more of the bass spectrum.

[source,text]
----
In 1
-> VCA Comp
-> Microtubes B3K
-> Amped Super Valve
-> 810 Amped VT 90s
-> Low-High Cut
-> Parametric-8
-> Gain
-> Out
----

== How this bass preset complements the guitars

[%autowidth,cols="1,1,3",options="header"]
|===
|Frequency area |Primary owner |Reason
|35 to 60 Hz |Bass |True low foundation. Guitars should not occupy this range heavily.
|70 to 100 Hz |Bass plus kick relationship |Main punch area. Adjust carefully with drums.
|120 to 250 Hz |Controlled overlap |Too much here makes the entire band muddy.
|700 Hz to 1.2 kHz |Bass growl |Lets the bass be heard through high-gain guitars.
|1.4 to 2.2 kHz |Bass note definition |Important for fast riff clarity.
|2.5 to 4 kHz |Controlled clank |Useful for pick attack, but too much becomes scratch.
|Above 6.5 kHz |Usually unnecessary |Often adds hiss and string noise more than musical information.
|===


== Correction workflow

[%autowidth,cols="2,2,3",options="header"]
|===
|Problem |Change |Reason
|Bass makes palm-muted guitars muddy |Raise Path B Low Cut to 200 Hz and cut final 240 Hz another -1 dB |Muddy guitars are often caused by bass low-mid overlap, not by the guitar presets.
|Bass has weight but no note definition |Raise Path B 1.15 kHz and 1.85 kHz by +1 dB |These bands carry growl and note identity through guitars.
|Bass sounds like a third guitar |Lower Path B Level to -5 dB and lower B3K Blend to 5.5 |Too much dirty path makes the bass compete with guitars instead of supporting them.
|Bass disappears under guitars |Raise Path B Level to -2.5 dB and boost 900 Hz or 1.6 kHz |Midrange makes bass audible on small speakers and through dense guitars.
|Low B or A is boomy |Reduce Path A 70 Hz boost and cut 105 Hz another -1 dB |Boom is usually low punch becoming low bloom.
|Low end is weak |Raise Path A Level +1 dB or add +0.5 to +1 dB around 70 to 85 Hz |Fix foundation on the clean-low path before adding dirty path.
|Bass is too clicky |Lower B3K Tone to 4.0 and reduce 2.9 kHz |Click comes from upper clank and drive tone.
|Dirty path is noisy during stops |Enable Utility Gate between B3K and Amped Super Valve using the tight-staccato settings |This cleans the distorted layer without choking the clean low path.
|Sustains or harmonics get cut off |Bypass the Utility Gate, or use Threshold -62 dB, Release 110 ms, Range -35 dB |Bass sustain is more important than guitar-style hard gating. The gate should only clean the dirty path when needed.
|===
