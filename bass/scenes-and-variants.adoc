== Scene A: Main rhythm

Use this for most riffs.

[%autowidth,cols="1,1",options="header"]
|===
|Section |Setting
|Crossover |180 Hz
|Path A Level |0.0 dB
|Path B Level |-3.5 dB
|B3K Drive |5.8
|B3K Blend |7.0
|Adaptive Gate |42%
|Final 85 Hz band |+1.2 dB
|===

== Scene B: Tight staccato / breakdown

Use this for dead stops, fast chugs, and very tight guitar sections.

[%autowidth,cols="1,1",options="header"]
|===
|Section |Setting
|Crossover |200 Hz
|Path A Level |0.0 dB
|Path B Level |-4.5 dB
|B3K Drive |5.2
|B3K Tone |4.5
|Adaptive Gate |50%
|Final 240 Hz band |-3.5 dB
|===

This scene should make the band feel tighter, not bigger.

== Scene C: Bigger chorus / open parts

Use this when the guitars are playing more open powerchords and the bass can occupy more space.

[%autowidth,cols="1,1",options="header"]
|===
|Section |Setting
|Crossover |160 Hz
|Path A Level |+1.0 dB
|Path B Level |-3.0 dB
|B3K Drive |5.5
|B3K Blend |6.5
|Adaptive Gate |35%
|Final 85 Hz band |+1.8 dB
|===

Do not use Scene C for the fastest riffs.

== Optional simpler stock-only version

Use this if the bassist does not want a parallel split.

[source,text]
----
In 1
-> VCA Comp
-> Microtubes B3K
-> Amped Super Valve
-> 810 Amped VT 90s
-> Low-High Cut
-> Parametric-8
-> Gain
-> Out
----

== Optional Parallax X version

Use this only if the bass player owns a Parallax X license on Quad Cortex.

[source,text]
----
In 1
-> VCA Comp
-> Parallax
-> Parametric-8
-> Gain
-> Out
----

Set Parallax by ear with strong low-band compression, main growl from the mid band, moderate high-band clank, cab on for mid/high bands, and low end kept independently tunable.
