== Purpose

This path is the clean foundation. It should sound stable, controlled, centered, and relatively plain by itself.

[source,text]
----
Path A:
Low-High Cut -> VCA Comp -> Parametric-8 -> Gain -> Mixer
----

== Low-High Cut

[%autowidth,cols="1,1",options="header"]
|===
|Parameter |Setting
|Low Cut |35 Hz
|High Cut |180 Hz
|===

== Tuning-specific low cut

[%autowidth,cols="1,1",options="header"]
|===
|Bass tuning |Low Cut
|E standard |40 Hz
|B standard |35 Hz
|Drop A |32 Hz
|Drop G |30 Hz
|Small PA / bad room |40 to 45 Hz
|===

== VCA Comp

[%autowidth,cols="1,1",options="header"]
|===
|Parameter |Setting
|Threshold |-26.0 dB
|Ratio |5.0:1
|Attack |30 ms
|Release |130 ms
|Makeup / Output |+3.0 dB
|Mix |85%
|===

== Low-path compressor fine-tuning

[%autowidth,cols="1,1",options="header"]
|===
|Problem |Change
|Low end pumps |Release 170 ms
|Low end feels slow |Release 100 ms
|Attack is too soft |Attack 22 ms
|Low B or A jumps out |Threshold -30 dB
|Low end is too flat |Mix 70%
|===

== Gain

[%autowidth,cols="1,1",options="header"]
|===
|Parameter |Setting
|Gain |0.0 dB
|===

Leave this at 0.0 dB initially. Use it only if you need to rebalance the clean low path against the dirty path.
