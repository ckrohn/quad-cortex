== Main correction table

[%autowidth,cols="1,1",options="header"]
|===
|Problem |Fix
|Bass makes palm-muted guitars muddy |Raise Path B Low Cut to 200 Hz and cut final 240 Hz another -1 dB
|Bass has weight but no note definition |Raise Path B 1.15 kHz and 1.85 kHz by +1 dB
|Bass sounds like a third guitar |Lower Path B Level to -5 dB and lower B3K Blend to 5.5
|Bass disappears under guitars |Raise Path B Level to -2.5 dB and boost 900 Hz and 1.6 kHz
|Low B / A is too boomy |Lower Path A 70 Hz boost and cut 105 Hz another -1 dB
|Low end is weak |Raise Path A Level +1 dB and boost 70 to 85 Hz
|Bass is too clicky |Lower B3K Tone to 4.0 and cut 2.9 kHz
|Bass is too dark |Raise B3K Tone to 5.5 and raise Path B Level
|Stops are noisy |Raise Adaptive Gate to 50%
|Sustains get cut off |Lower Adaptive Gate to 30% or bypass it
|Bass and guitars fight in low mids |Cut final 240 Hz and Path B 220 Hz
|===
