== Purpose

This keeps the 6505 identity but shifts the tone toward sustain, upper-mid focus, and controlled ambience. Path A is active and Path B is muted.

[source,text]
----
Path A:
Green 808 -> PV-505 Lead -> Adaptive Gate (S/C) -> IR Loader: Single (M)

Post-mixer:
Low-High Cut -> Parametric-8 -> Digital Delay (M) -> Plate -> Gain -> Output
----

== Lead correction table

[%autowidth,cols="1,1",options="header"]
|===
|Problem |Change
|Lead sustain gets chopped |Lower Adaptive Gate to 70% to 72%
|Lead delay clouds fast runs |Delay Mix 8% to 10%, Feedback 20% to 24%
|Lead is buried |Raise final Gain by +0.5 to +1.0 dB or add +1 dB at 1.25 kHz
|Lead is fizzy |Lower High Cut to 7.2 to 7.5 kHz or cut 3.75 kHz another -1 dB
|===
