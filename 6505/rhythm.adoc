== Purpose

This is the main dry 6505 rhythm scene. Path A is active and Path B is muted at the Mixer.

[source,text]
----
Path A:
Green 808 -> PV-505 Lead -> Adaptive Gate (S/C) -> IR Loader: Single (M)

Post-mixer:
Low-High Cut -> Parametric-8 -> Gain -> Output
----

== Rhythm correction table

[%autowidth,cols="1,1",options="header"]
|===
|Problem |Change
|Palm mutes are muddy |Lower Resonance to 1.2, raise Low Cut to 88 Hz, or cut 220 Hz another -1 dB
|Tone is fizzy |Lower High Cut to 7.4 to 7.6 kHz or lower Presence to 3.4
|Powerchords smear |Lower Gain to 2.9 and cut 360 Hz another -1 dB
|Tone disappears with the full band |Raise Mid to 6.3 or boost 1.25 kHz another +0.8 dB
|===
