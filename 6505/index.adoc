== Global preset structure

Build this as one Quad Cortex preset with a dirty path and a clean path. Use scene-specific Mixer levels and scene-specific parameter values.

[source,text]
----
In 1
-> Splitter: A/B

Path A, dirty rhythm/lead:
Green 808 -> optional Simple Gate -> PV-505 Lead -> Adaptive Gate (S/C) -> IR Loader: Single (M)

Path B, clean:
VCA Comp -> Rols Jazz CH120 -> 212 Rols Jazz '87

Mixer
-> Low-High Cut -> Parametric-8 -> Chief CE2W (M) -> Digital Delay (M) -> Plate -> Gain -> Output
----

== Fast corrections

[%autowidth,cols="1,1",options="header"]
|===
|Problem |Change
|Rhythm palm mutes muddy |Lower Resonance to 1.2, raise Low Cut to 88 Hz, or cut 220 Hz another -1 dB
|Rhythm fizz |Lower High Cut to 7.4 to 7.6 kHz or lower Presence to 3.4
|Clean is boomy |Raise clean Low Cut to 125 to 135 Hz
|Clean chorus is too obvious |Chief CE2W Mix 18% and Depth 30%
|Lead sustain gets chopped |Adaptive Gate 70% to 72%
|Lead delay clouds fast runs |Delay Mix 8% to 10%, Feedback 20% to 24%
|Lead buried |Raise final Gain by +0.5 to +1.0 dB or add +1 dB at 1.25 kHz
|===

Keep rhythm scenes dry. Use delay and reverb only in clean and lead scenes. Mute the inactive path at the Mixer, not only by bypassing blocks.
