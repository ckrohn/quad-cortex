== Purpose

This is the brighter and wider clean guitar. Path B is active and Path A is muted at the Mixer.

[source,text]
----
Path B:
VCA Comp -> Rols Jazz CH120 -> 212 Rols Jazz '87

Post-mixer:
Low-High Cut -> Parametric-8 -> Chief CE2W (M) -> Digital Delay (M) -> Plate -> Gain -> Output
----

== Clean correction table

[%autowidth,cols="1,1",options="header"]
|===
|Problem |Change
|Clean is boomy |Raise Low Cut to 125 to 135 Hz
|Clean is too thin |Lower Low Cut to 95 to 105 Hz
|Clean clips with Fishmans |Lower Rols Jazz Volume by 0.5 to 1.0
|Chorus is too obvious |Chief CE2W Mix 18% and Depth 30%
|Delay/reverb is too wet |Reduce Delay Mix to 8% to 10% and Plate Mix to 10% to 12%
|===
