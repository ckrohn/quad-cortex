== Target

The EV101 side supplies width, polish, and a smoother Mesa Oversized-style cab body. It should complement the 6505 side, not copy it. The rhythm scene is intentionally dry: no chorus, no delay, no reverb, no doubler, and no stereo widening.

== Active routing

[source,text]
----
Path A active:
Green 808 -> Utility Gate -> EV101IIIS Red 6L6 100W -> IR Loader: Single (M)

Path B muted at Mixer.

Post Mixer:
Low-High Cut -> Parametric-8 -> Gain -> Output
----

== Green 808

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|Overdrive |0.00 |The boost is for tightening the amp input, not for adding pedal distortion.
|Tone |5.8 |Brighter than the 6505 side to keep the smoother OS-V2 57 + 121 mix present in the band.
|Level |7.8 |A firm push into the EV101 model tightens palm mutes and improves pick response.
|Bypass |Off |The 808 is part of the rhythm voicing and should stay on.
|===


The overdrive is doing pre-EQ work. It reduces loose low-end before the amp and emphasizes the attack range. For this style, that is usually more useful than adding more amp gain.

== Utility Gate

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|Threshold |-53.0 dB |This matches the working setting shown on the QC. It is high enough for hard modern stops with hot Fishmans and an 808 boost, but not so high that normal picked rhythm notes should fade in.
|Attack |1 ms |Fastest practical opening. This removes the fade-in behavior heard with the Adaptive Gate and preserves the front edge of palm mutes.
|Hold |15 ms |Keeps the gate open briefly after the transient so it does not chatter between the initial pick attack and the start of note sustain.
|Release |38 ms |Fast enough for tight metalcore/deathcore stops. This is intentionally short for rhythm and would be too abrupt for many lead parts.
|Range |-70.0 dB |Hard attenuation for dry rhythm silence. This gives the sharp stop behavior expected from modern gated rhythm guitars.
|Bypass |Off |The gate is part of the rhythm feel.
|===

Place this block between `Green 808` and the amp, not after the amp. The 808 gives the gate a more consistent level to read, while the amp still receives a tightly muted signal. Placing the gate before the 808 can work, but the 808 level boost will then amplify any remaining pickup noise after the gate. Placing a non-sidechained gate after a high-gain amp usually tracks the compressed distorted signal less accurately and can chop harmonics or feel disconnected.

If harmonics or sustained powerchords are cut off, lower Threshold to `-56.0 dB` or `-58.0 dB` before changing Attack or Release. If dead stops are still loose, raise Threshold to `-51.0 dB` or shorten Release to `30 ms`.

== EV101IIIS Red 6L6 100W

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|Gain |3.8 |Moderate high gain. Higher settings compress the transient and blur low-string riffs.
|Bass |2.4 |Enough body for the larger OS-V2 cab, but not enough to fight bass guitar.
|Mid |5.7 |Keeps the EV101 side present while allowing the 6505 side to be the more forward midrange voice.
|Treble |5.1 |Adds articulation through the smoother 57 + 121 IR.
|Presence |4.5 |Adds high-end cut without relying on excessive treble.
|Resonance / Depth |2.0 |Gives some low-end authority, but stays controlled for a 7-string.
|Master |3.5 |A moderate value for body and sustain without excessive modeled power-amp bloom.
|Level |Unity, usually -4.0 dB to -2.0 dB |Use this only for block level control, then final Gain for scene matching.
|===


The low gain and restrained low-end controls are intentional. With a 7-string and Fishman Fluence Modern pickups, brutality comes from tight transient shape and bass-guitar support, not from excessive guitar sub-low content.

== Post-mixer Low-High Cut

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|Low Cut |82 Hz |Tightens the larger OS-V2 cab while keeping the EV101 side slightly bigger than the 6505 side.
|High Cut |8.4 kHz |Leaves a little more top than the 6505 side because the 57 + 121 mix is smoother.
|===


A guitar low cut can feel counterintuitive when playing alone, but it is critical in a full band. It removes rumble that the bass and kick handle more effectively.

== Post-mixer Parametric-8

[%autowidth,cols="1,1,1,1,1,3",options="header"]
|===
|Band |Type |Frequency |Gain |Q |Purpose
|1 |Bell |120 Hz |-1.8 dB |1.00 |Controls oversized-cab low-end mass.
|2 |Bell |240 Hz |-2.8 dB |1.10 |Main low-mid mud cut for this side.
|3 |Bell |420 Hz |-1.2 dB |1.00 |Keeps powerchords from becoming woolly.
|4 |Bell |850 Hz |+1.0 dB |0.80 |Adds chord body and growl.
|5 |Bell |1.45 kHz |+1.5 dB |0.90 |Restores forwardness that can be softened by the 121 component.
|6 |Bell |2.60 kHz |+1.2 dB |1.00 |Adds pick attack and separation.
|7 |Bell |3.90 kHz |-1.0 dB |1.80 |Controls harshness while leaving bite.
|8 |Bell |6.30 kHz |-0.8 dB |1.40 |Smooths active-pickup scratch.
|===


The EQ is mild on purpose. Extreme scooping often sounds impressive alone and weak in a two-guitar band. These moves remove mud, retain midrange identity, and control harshness.

== Rhythm correction workflow

[%autowidth,cols="2,2,3",options="header"]
|===
|Problem |Change |Reason
|Palm mutes are muddy |Lower amp resonance/depth first, then raise Low Cut by 4 to 6 Hz |Preamp and power-amp low-end controls affect the mud source before post-EQ cleanup.
|Powerchords smear |Lower amp Gain by 0.3 to 0.7 and cut the 220 to 420 Hz range another -0.5 to -1.0 dB |Too much saturation and low-mid buildup hide chord intervals.
|Tone is fizzy |Lower High Cut to 7.4 to 7.8 kHz or cut the 3.7 to 3.9 kHz band another -1 dB |This removes harshness without making the whole amp darker.
|Tone disappears with the band |Add +0.5 to +1.0 dB around 1.2 to 1.6 kHz |That range carries riff identity through bass and drums.
|Pick attack is too clicky |Lower Green 808 Tone by 0.4 to 0.8 and reduce the 2.3 to 2.6 kHz boost |Click usually comes from boost tone plus upper-mid EQ, not from presence alone.
|Dead stops are loose |Raise Utility Gate Threshold to -51.0 dB or shorten Release to 30 ms |Threshold is the primary tightness control. Use the smallest change that gives clean stops.
|Harmonics or held chords disappear |Lower Utility Gate Threshold to -56.0 dB or -58.0 dB |This lets lower-level sustain remain above the gate threshold.
|Gate chatters on sustained notes |Raise Hold to 20 ms or Release to 50 ms |Hold and release smooth the closing behavior without making the gate open slower.
|===
