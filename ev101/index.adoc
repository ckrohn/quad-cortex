== Global preset structure

Build this as one Quad Cortex preset with a dirty path and a clean path. Use scene-specific Mixer levels and scene-specific parameter values.

[source,text]
----
In 1
-> Splitter: A/B

Path A, dirty rhythm/lead:
Green 808 -> optional Simple Gate -> EV101IIIS Red 6L6 100W -> Adaptive Gate (S/C) -> IR Loader: Single (M)

Path B, clean:
VCA Comp -> US DLX 65 Reissue -> 112 US DLX SC64

Mixer
-> Low-High Cut -> Parametric-8 -> Chief CE2W (M) -> Digital Delay (M) -> Plate -> Gain -> Output
----

== Fast corrections

[%autowidth,cols="1,1",options="header"]
|===
|Problem |Change
|Rhythm palm mutes muddy |Lower Resonance / Depth to 1.5, raise Low Cut to 86 Hz, or cut 240 Hz another -1 dB
|Rhythm too smooth |Add +0.5 to +1.0 dB at 1.45 kHz or 2.60 kHz
|Clean is boomy |Raise clean Low Cut to 130 Hz and lower Bass to 3.0
|Clean is too bright |Lower clean High Cut to 8.0 kHz and lower Treble to 4.5
|Lead sustain gets chopped |Adaptive Gate 70% to 72%
|Lead delay clouds harmonized leads |Delay Mix 8% to 10%, Feedback 20% to 24%
|Lead buried |Raise final Gain by +0.5 to +1.0 dB or add +1 dB at 1.45 kHz
|===

Keep the EV101 side smoother and larger than the 6505 side. The 6505 side supplies sharper 421 + 57 attack. The EV101 side supplies width and smoother sustain.
