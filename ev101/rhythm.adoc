== Purpose

This is the polished, tight EV101 side. Path A is active and Path B is muted at the Mixer.

[source,text]
----
Path A:
Green 808 -> EV101IIIS Red 6L6 100W -> Adaptive Gate (S/C) -> IR Loader: Single (M)

Post-mixer:
Low-High Cut -> Parametric-8 -> Gain -> Output
----

== Rhythm correction table

[%autowidth,cols="1,1",options="header"]
|===
|Problem |Change
|Palm mutes are muddy |Lower Resonance / Depth to 1.5, raise Low Cut to 86 Hz, or cut 240 Hz another -1 dB
|Tone is too smooth |Add +0.5 to +1.0 dB at 1.45 kHz or 2.60 kHz
|Tone is fizzy |Lower High Cut to 7.8 kHz or cut 3.90 kHz another -1 dB
|Pick attack is too clicky |Lower Green 808 Tone to 5.0 and reduce 2.60 kHz by -1 dB
|===
