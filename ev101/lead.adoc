== Purpose

This is the smoother EV101 lead scene with more mids, less resonance, and controlled delay/reverb. Path A is active and Path B is muted.

[source,text]
----
Path A:
Green 808 -> EV101IIIS Red 6L6 100W -> Adaptive Gate (S/C) -> IR Loader: Single (M)

Post-mixer:
Low-High Cut -> Parametric-8 -> Digital Delay (M) -> Plate -> Gain -> Output
----

== Lead correction table

[%autowidth,cols="1,1",options="header"]
|===
|Problem |Change
|Lead sustain gets chopped |Lower Adaptive Gate to 70% to 72%
|Lead delay clouds harmonized leads |Delay Mix 8% to 10%, Feedback 20% to 24%
|Lead is buried |Raise final Gain by +0.5 to +1.0 dB or add +1 dB at 1.45 kHz
|Lead is fizzy |Lower High Cut to 7.4 to 7.6 kHz or cut 3.90 kHz another -1 dB
|===
