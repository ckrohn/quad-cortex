== Target

The 6505 player clean scene is the brighter and wider clean texture. It uses more chorus than the EV101 side so the two clean tones do not collapse into one identical sound. The clean scene uses Path B only. It should support atmospheric parts without putting uncontrolled low-end under the bass.

== Active routing

[source,text]
----
Path A dirty path muted at Mixer.

Path B active:
VCA Comp -> Rols Jazz CH120 -> 212 Rols Jazz '87

Post Mixer:
Low-High Cut -> Parametric-8 -> Chief CE2W (M) -> Digital Delay (M) -> Plate -> Gain -> Output
----


== Mixer state

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|Level A |-60.0 dB |Mutes the high-gain path so no distorted sound leaks into clean.
|Level B |0.0 dB |Clean path is active.
|Pan A / Pan B |C / C |Mono routing keeps both guitarists predictable live.
|Phase |Off |Normal phase.
|Mixer Level |0.0 dB |No scene volume change here.
|===


== VCA Comp

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|Threshold |-24.0 dB |Set low enough to catch uneven clean peaks from active pickups.
|Ratio |3.0:1 |Moderate compression for even clean chords without making the guitar feel flat.
|Attack |14 ms |Lets the initial pick transient through, then smooths the body of the note.
|Release |110 ms |Returns naturally between clean chord hits without obvious pumping.
|Makeup |+2.5 dB |Restores level lost to compression so the clean path does not feel weak.
|Mix |75% |Keeps most of the compressed signal while retaining some natural touch.
|Bypass |Off |Compression is part of the clean scene foundation.
|===


The clean compressor controls active-pickup peaks and makes arpeggiated parts more even. It is not meant to squash the guitar into a pop clean tone. Keep enough attack for pick definition.

== Rols Jazz CH120

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|Volume |2.2 |Keeps the Rols Jazz clean even with hot pickups. Higher values can make the clean scene bark or clip.
|Bass |3.8 |Enough fullness for clean chords, but not enough to fight the bass guitar.
|Middle |5.0 |Neutral midrange. The post-EQ provides the final band fit.
|Treble |5.8 |Adds the glassy clean top that contrasts the dry rhythm channel.
|Level |0.0 dB |Initial clean amp level. Final scene volume is set with the final Gain block.
|Bypass |Off |This is the active clean amp when Path B is selected.
|===


This clean amp is chosen because it gives a separate identity from the high-gain amp. Cleaning up a high-gain amp block is less predictable than using a real clean path.

== Post-mixer Low-High Cut

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|Low Cut |115 Hz |Removes low-string bloom from clean parts and leaves the bass stable.
|High Cut |9.2 kHz |Keeps clean sparkle while preventing harsh active-pickup top.
|===


The clean low cut is higher than the rhythm low cut because clean chords have more sustained low-frequency energy. That sustained low-end can mask the bass more than short distorted palm mutes.

== Post-mixer Parametric-8

[%autowidth,cols="1,1,1,1,1,3",options="header"]
|===
|Band |Type |Frequency |Gain |Q |Purpose
|1 |Bell |120 Hz |-2.0 dB |1.00 |Further reduces clean low-end weight so open chords do not cloud bass.
|2 |Bell |240 Hz |-2.5 dB |1.10 |Main clean mud cut. This is important with wide clean chorus.
|3 |Bell |420 Hz |-1.0 dB |1.00 |Removes boxy low-mid buildup.
|4 |Bell |750 Hz |+0.6 dB |0.90 |Adds enough body so clean chords are not hollow.
|5 |Bell |1.20 kHz |+1.0 dB |0.90 |Helps the clean line stay audible under cymbals.
|6 |Bell |2.20 kHz |+0.8 dB |1.00 |Adds pick clarity without making the clean tone clicky.
|7 |Bell |3.40 kHz |-1.0 dB |1.50 |Softens hard active-pick attack.
|8 |Bell |6.80 kHz |-0.5 dB |1.30 |Subtle smoothing above the chorus/delay range.
|===


The clean EQ is designed to remove boom, keep chord detail, and control active-pickup attack. The mid boosts are modest because the clean scene should sit in the mix, not dominate it unless the arrangement requires it.

== Chief CE2W (M)

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|Mix |24% |Enough effect to make this the wider clean guitar, but still below obvious 1980s-style wet chorus.
|Rate |18% |Slow movement. Faster rates sound seasick and distract from clean picking.
|Sync |Off |The chorus should float independently rather than pulse with song tempo.
|Sync Note |Ignored when Sync is Off |No tempo subdivision is used in this scene.
|Depth |38% |Moderate depth gives width without pulling chords out of tune.
|Type |CE2-C |A fuller chorus voicing that suits the brighter Rols Jazz clean sound.
|Out Mode |Mono |The preset remains mono so it translates through front of house and does not fight the second guitarist.
|Output |0.0 dB |No level change from the chorus block. Scene volume is controlled at the end.
|Bypass |Off |This scene uses chorus as part of the clean identity.
|===


The two guitarists use different chorus depths and mixes. This keeps the clean section wide and layered without both guitars producing the same modulation pattern.

== Digital Delay (M)

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|Mix |12% |Audible space without turning the clean rhythm into delay repeats.
|Feedback |24% |Enough repeats for ambience, but not enough to blur chord changes.
|High Pass |180 Hz |Removes low-end from repeats so delay does not smear the bass or low strings.
|Low Pass |4500 Hz |Darkens repeats so the dry clean attack remains in front.
|Sync |Off |Uses a fixed delay time for separation from the second guitar.
|Sync Note |Ignored when Sync is Off |No tempo subdivision is active.
|Delay Time |420 ms |Long enough to create space, different from the EV101 side to avoid stacked identical repeats.
|Mod Rate |0.35 Hz |Very slow movement in the repeats.
|Mod Depth |16% |Subtle modulation keeps repeats from sounding sterile without replacing the chorus.
|Trails |On |Lets delay decay naturally when changing scenes.
|Bypass |Off |Delay is part of the clean scene.
|===


The delay is filtered and relatively quiet. Low-passing the repeats keeps them behind the dry guitar, and high-passing the repeats prevents low-end buildup.

== Plate

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|Mix |16% |More spacious than the lead scene but still controlled for a metal band mix.
|Decay |70% |Long enough for clean ambience without turning into a pad.
|Pre Delay |25.0 ms |Keeps the dry pick attack clear before the reverb bloom begins.
|High Pass |200 Hz |Removes low-frequency reverb build-up.
|Low Pass |3100 Hz |Keeps the reverb dark so it does not fight cymbals or pick attack.
|LF Damping |45% |Moderate damping so the lower reverb body does not hang too long.
|HF Damping |62% |Stronger high damping for a smoother tail.
|Mod Depth |18% |Small modulation prevents a static reverb tail.
|Trails |On |Lets the reverb decay naturally after scene changes.
|Bypass |Off |Reverb is part of the clean scene.
|===


The plate reverb is darker and filtered. Bright, full-range reverb makes clean guitars sound impressive alone but can wash over vocals, cymbals, and lead melodies in a dense metal arrangement.

== Final Gain

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|Gain |+3.5 dB |Clean sounds are less compressed than high-gain rhythm, so they usually need a final level lift. Match this at band volume.
|Bypass |Off |Use this block for clean scene volume instead of compressor makeup or amp level.
|===


== Clean correction workflow

[%autowidth,cols="2,2,3",options="header"]
|===
|Problem |Change |Reason
|Clean is boomy |Raise Low Cut to 125 to 135 Hz and reduce the 240 Hz band by another -0.5 to -1 dB |Sustained clean low end is the usual cause.
|Clean is too thin |Lower Low Cut to 100 to 110 Hz and add +0.5 dB around 700 to 900 Hz |This restores body without adding sub-low clutter.
|Clean clips or crunches |Lower clean amp Volume by 0.5 to 1.0 |Hot active pickups can push clean amps harder than expected.
|Chorus is too obvious |Lower Chief CE2W Mix by 4 to 8 percentage points and reduce Depth |Chorus depth and mix create the perception of pitch movement.
|Delay or reverb clouds the part |Lower Delay Mix to 8 to 10% and Plate Mix to 9 to 12% |Ambience should support the clean part, not become the part.
|===
