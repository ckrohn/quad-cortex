== Role

Tighter, nastier, more forward guitar side. This preset is designed for a 7-string guitar with hot active Fishman Fluence Modern pickups in a two-guitar metalcore/deathcore band.

== Scene documents

[%autowidth,cols="1,2,3",options="header"]
|===
|Scene |File |Purpose
|Scene A: Rhythm |link:rhythm.adoc[rhythm.adoc] |Dry modern rhythm tone with tight palm mutes and no time effects.
|Scene B: Clean |link:clean.adoc[clean.adoc] |Dedicated clean path with compression, clean amp/cab, chorus, delay, and plate reverb.
|Scene C: Lead |link:lead.adoc[lead.adoc] |High-gain lead tone with more mids, less gate, delay, plate, and a final volume lift.

== Global routing

Build this as one Quad Cortex preset with a dirty path and a clean path. The scene change is handled by the Mixer, not by relying only on bypass states.

[source,text]
----
In 1
-> Splitter: A/B

Path A, dirty rhythm/lead:
Green 808 -> Utility Gate -> PV-505 Lead -> IR Loader: Single (M)

Path B, clean:
VCA Comp -> Rols Jazz CH120 -> 212 Rols Jazz '87

Mixer
-> Low-High Cut
-> Parametric-8
-> Chief CE2W (M)
-> Digital Delay (M)
-> Plate
-> Gain
-> Output
----


== Input settings

[%autowidth,cols="1,1,3",options="header"]
|===
|Parameter |Setting |Why this value is used
|Input Gain |-6.0 dB |Hot active pickups can overload the first block and make the amp feel congested. Start at -6.0 dB and adjust only if the input meter requires it.
|Input Gate |Off |Use the dedicated gate later in the chain. The input gate is too global for a preset that has clean, rhythm, and lead scenes.
|===


== Gate architecture

The dirty path uses `Utility Gate`. The gate sits between the boost and the amp:

[source,text]
----
Green 808 -> Utility Gate -> high-gain amp
----

This position is intentional. The 808 shapes and stabilizes the signal before the gate, and the amp receives a cleanly muted input. A gate before the 808 can leave noise that the 808 level control then amplifies. A non-sidechained gate after a high-gain amp has to read a heavily compressed distorted signal, which can cause fade-in, unnatural chopping, or lost harmonics.

[%autowidth,cols="1,1,1,3",options="header"]
|===
|Scene |Threshold |Release |Reason
|Rhythm |-53.0 dB |38 ms |Hard, fast stop behavior for palm-muted rhythm.
|Lead |-60.0 dB |90 ms |Lower threshold and longer release for bends, vibrato, and harmonics.
|Clean |Not used |Not used |The dirty path is muted. The clean path should remain ungated unless the instrument is unusually noisy.
|===

Use threshold as the first correction control. Raise it for tighter stops. Lower it if harmonics, slides, or held notes disappear.

== Mixer scene switching

[%autowidth,cols="1,1,1,1,3",options="header"]
|===
|Mixer parameter |Rhythm |Clean |Lead |Why
|Level A |0.0 dB |-60.0 dB |0.0 dB |Path A is dirty rhythm/lead. Muting it in clean prevents high-gain leakage.
|Pan A |C |C |C |Both guitar presets are mono for consistency and FOH translation.
|Level B |-60.0 dB |0.0 dB |-60.0 dB |Path B is clean. It is muted in rhythm and lead scenes.
|Pan B |C |C |C |No artificial stereo spread is used in one player preset.
|Phase |Off |Off |Off |Use normal phase unless a specific cab or split-path phase issue is heard.
|Mixer Level |0.0 dB |0.0 dB |0.0 dB |Scene volume is handled at the final Gain block instead.
|===


If the Quad Cortex does not show exactly `-60.0 dB`, set the inactive path as low as the device allows. The practical target is silence from the inactive path.

== Volume matching workflow

. Match rhythm scenes first with both guitarists playing the same riff.
. Add bass and confirm palm mutes get heavier, not blurrier.
. Set clean scenes slightly below or equal to rhythm unless the song needs cleans to dominate.
. Set lead scenes around `+2.5 dB` to `+3.5 dB` over rhythm using final Gain only.
. Re-check at rehearsal volume because high-gain compression can make bedroom-level balances misleading.

== Fast global corrections

[%autowidth,cols="2,2,3",options="header"]
|===
|Problem |Change |Reason
|Input clips on hard palm mutes |Lower Input Gain to -9.0 dB |Input clipping creates harshness and mud before any block can fix it.
|Clean and dirty path leak together |Check Mixer Level A/B for every scene |Bypassing blocks alone is not as reliable as muting the inactive path at the Mixer.
|Rhythm sounds large alone but messy with bass |Reduce 220 to 360 Hz and keep the low cut where specified |Low-mid buildup is the usual cause of full-band mud.
|Lead is not loud enough |Raise final Gain by +0.5 dB to +1.0 dB |This changes level without changing amp saturation.
|Scene changes leave too much ambience |Turn Trails Off on delay/reverb for songs with hard stops |Trails are musical for transitions but can conflict with dead stops.
|===
